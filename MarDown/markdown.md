## Aula Markdown

Use # para criar titulos

# Titulo 1
## Titulo 2
### Titulo 3
#### Titulo 4
##### Titulo 5
###### Titulo 6

---

Use sinal ( - ) ou ( * ) para criar topicos

* Topico (*)
- Topico (-)

---

Use um * para deixar em italico e dois ** para deixar em negrito, e use ~~ oara taxado.

*Italico*

**Negrito**

~~Taxado~~

---

Para criar listas numeradas use o numero seguido de um ponto " . "

- item
    - item
    - item
- item
- item
